# 📝 To-Do App

## 📌 项目简介（Project Overview）

这是一个使用 HTML / CSS / JavaScript 开发的待办事项管理应用。
This is a simple To-Do application built with HTML, CSS, and JavaScript.

用户可以：

* 添加任务
* 删除任务
* 标记任务完成状态
* 按状态筛选任务（All / Active / Completed）
* 刷新页面后数据不会丢失

---

## 🚀 功能（Features）

* Task Add / Delete
* Task Completion Toggle
* Filter (All / Active / Completed)
* LocalStorage 数据持久化
* UI 简洁设计

---

## 🛠 技术栈（Tech Stack）

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 💡 项目亮点（Highlights）

* 使用 localStorage 实现数据持久化
* 实现任务状态管理（completed / active）
* 使用 DOM 操作实现动态渲染
* 实现筛选功能（条件渲染）

# todo-app

讲解项目：这是一个我独立开发的前端 To-Do 应用，主要使用原生 JavaScript 实现。

项目的核心在于：
第一，实现了任务的增删改查功能；
第二，通过 localStorage 实现了数据持久化；
第三，设计了任务的完成状态，并支持筛选功能；
第四，通过 DOM 操作实现动态渲染。

在开发过程中，我重点提升了对 JavaScript 状态管理和数据与UI同步的理解。
